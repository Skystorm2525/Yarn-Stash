Yarn Stash

Yarn Stash is a Flask web app for managing:
	•	Yarn inventory
	•	Crochet and knitting patterns
	•	Projects
	•	Pattern folders

Features:
	•	Upload PDF or image patterns
	•	Organize patterns into folders
	•	Assign patterns to projects
	•	Track skeins owned and allocated
	•	Clean dashboard view

Tech Stack:
	•	Python
	•	Flask
	•	SQLite
	•	HTML / CSS
